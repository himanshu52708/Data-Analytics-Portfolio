# Retail Sales & Customer Behavior Analysis

## Project Overview

This project analyzes retail sales and customer behavior using SQL (MySQL Workbench).

The objective was to identify:

* Revenue opportunities
* Customer purchasing patterns
* Product performance
* Profitability trends

## Tools Used

* SQL
* MySQL Workbench
  
## Dataset

* Orders Table
* Customers Table

## Key Analysis

* Revenue by State
* Revenue by Product Category
* Customer Segmentation
* Revenue by Education
* Revenue by Marital Status
* Revenue by Age Group
* Top Customers Analysis
* Profitability Analysis

## Key Findings

* California generated approximately 40% of total revenue.
* Household Appliances were the highest-performing category.
* TV products generated the largest share of revenue.
* Adults aged 30–50 represented the highest-spending customer segment.
* College-educated customers contributed the highest revenue.

## Business Recommendations

* Increase investment in California and Washington markets.
* Expand inventory for high-performing TV products.
* Create loyalty programs for high-value customers.
* Target marketing toward high-performing customer segments.
