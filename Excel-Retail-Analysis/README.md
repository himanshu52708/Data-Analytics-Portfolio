# Excel Retail Sales Analysis

## Overview

Analyzed retail sales data using Microsoft Excel to identify revenue trends, customer behavior, and product performance.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

## Key Insights

* Identified top-performing products and categories.
* Analyzed customer purchasing patterns.
* Evaluated revenue contribution across segments.
* Created an interactive dashboard for business reporting.

## Dashboard Features

* Total Revenue KPI
* Total Profit KPI
* Customer Analysis
* Product Performance Analysis
* Interactive Filters and Slicers
